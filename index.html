About Me

I’ve always been interested in learning about software development and web design, but I also have interests in application development and video game design. I completed an internship with Penn State’s Chemistry Department in which I developed 2 JavaScript applications for a class eBook. I have experience in several programming languages including HTML, CSS + Bootstrap, JavaScript + jQuery, PHP, SQL, and Java.
